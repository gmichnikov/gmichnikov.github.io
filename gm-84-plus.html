GM-84 Plus

A browser-based graphing calculator inpired by the legendary Texas Instruments products

GM-84 Plus is a JavaScript graphing calculator built using jQuery, HTML5 Canvas, Math.js, and Mathquill. It replicates the basic graphing features of a standard graphing calculator: graphing any equation the user enters in y=f(x) form; zooming/panning to anywhere in the xy-plane to allow for examination of small details or asymptotic behavior; setting the window to standard or custom values; and tracing the values of y as x moves from the window's minimum to its maximum.

The biggest addition in the GM-84 Plus is the ability to animate a graph to observe how it changes as a constant within the equation moves between user-defined minimum and maximum values. By animating the graphs, users can get a better understanding of algebraic transformations. Plus, they're fun to look at! Users can also manually animate the graphs using the slider at the bottom of the screen, and they can choose colors for each graph (or hide each graph) in the panel to the left.

Both the zooming and panning features were custom-built for this application. Users can zoom in on any point in the plane, at which point the GM-84 Plus will use dilations to recalculate the window measurements and re-draw each graph. To enable drag-and-drop panning, the GM-84 Plus constantly "listens" for mouse clicks and mouse movement; it then adjusts the window and redraws the graphs accordingly. You can try out the calculator or browse the code on Github.
