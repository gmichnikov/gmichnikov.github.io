Untrippd Live Github

A fullstack, single-page travel recommendation sharing app, inspired by Untappd

Untrippd is a website built using Ruby, Rails, PostgreSQL, JavaScript, React, and Redux. It allows users to harness the best possible source of travel advice: friends who have been there before.

Users can easily share recommendations with their friends, so that others can benefit from their travel experiences; they can also browse others' recommendations, discovering tidbits that will make their own trips that much more memorable.

In addition to leaving suggestions, users can follow other users, like suggestions made by other users, and filter the suggestion feed by follows, likes, and tags (e.g. food or highlight). Untrippd uses Amazon Web Services to allow photo uploads along with each suggestion, and it uses the Google Places Auotcomplete API to allow users to tag every suggestion with any address, business, or other location known to Google.

Under the hood, Untrippd optimizes aggregation of suggestions (by country, or state) using a polymorphic Suggestion model and custom SQL to avoid unnecessary database queries. Untrippd aims to be as DRY and composable as possible, re-using partials and React components every time the Suggestion Feed is displayed throughout the site. You can check out the live site using the guest login, or sign up for an account and start sharing suggestions of your own! All of the code can be viewed on Github.
